Killer Carousel Example: Banner

As well as fancy 3D style carousels, Killer Carousel allows you to create 2D HTML slides, especially useful for large 'hero' style banners.

Just define a banner of some width, in this case width:800, set spacing to match, spacing2d:800 and set autoScale:100 to make it expand to 100% width.

Killer Carousel will then implement a swipe response more suited to banners.
