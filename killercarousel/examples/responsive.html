Killer Carousel Example: Responsive carousel

This is a responsive carousel that changes size with the parent element.

As in Example 1, the carousel is initially defined as 800 pixels wide, but it uses the autoScale property to set the carousel size to 75% of the parent element width.

Examples of the scaled width of the carousel are as follows:

  • Parent = 600, carousel = 450
  • Parent = 1067, carousel = 800 (unaltered scale)
  • Parent = 1280, carousel = 960

Notice how the carousel elements correctly scale in proportion to the width.

The infiniteLoop property is used to make the carousel loop forever.
